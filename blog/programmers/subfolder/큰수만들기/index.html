    완전탐색 (실패)

    k 개를 뺀 모든 숫자를 비교한 뒤 가장 큰 값을 리턴. 재귀 함수를 사용. 시간 초과로 실패.

    def makeNum(num, j, k, maxNum):
    if j == k:
        return int(num)

    for i in range(len(num)):
        temp = num[:i] + num[i + 1:]
        maxNum = max(maxNum, makeNum(temp, j + 1, k, maxNum))
        
    return maxNum

def solution(number, k):
    return str(makeNum(number, 0, k, 0))

    스택 LIFO (성공)

    핵심은 스택의 마지막 값이 push 할 값보다 작다면 크거나 같은 값이 나올 때까지 값들에 대해서 pop을 하는 것입니다. 이렇게 풀이하면 O(n)의 시간 복잡도로 문제를 해결할 수 있습니다.

    풀이 예시

    제거 횟수를 다 사용하지 않았을 때 남은 횟수만큼 리스트를 잘라줍니다. len(number) - k 까지 슬라이싱 해줍니다.

    def solution(number, k):
    stack = []
    
    for i in range(len(number)):
        num = int(number[i])

        while k > 0 and stack and stack[-1] < num:
            stack.pop()
            k -= 1
            
        stack.append(num)
    
    res = "".join(map(str, stack))
    return res[:len(res) - k]
